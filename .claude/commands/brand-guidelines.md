## Voice & Tone
- **Approachable** — speak like a knowledgeable friend, not a corporation
- **Encouraging** — learning is a journey; celebrate progress
- **Clear** — no jargon unless the audience expects it
- **Inclusive** — language that welcomes all backgrounds and ages

**Avoid:** passive voice, filler words ("very", "just", "really"), overpromising.

---

## Logo Usage
- Minimum clear space: equal to the height of the "T" in TWH around all sides
- Never stretch, rotate, or recolor the logo
- Use the light version on dark backgrounds, dark version on light backgrounds

---

## UI Principles for TWH
1. Educational content is the hero — UI should never compete with it
2. Progress and achievement should be visible and celebrated
3. Navigation should feel effortless — learners are focused on content, not tools
4. Trust signals matter: credentials, reviews, and instructor bios should be prominent
